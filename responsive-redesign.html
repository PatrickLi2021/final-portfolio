Responsive Redesign


Revamping the Lower Providence Community Library Website

Context and Overview:


The goal of this project was to redesign the Lower Providence Community Library website to address usability issues and create a more user-friendly interface. The original website had several flaws related to navigation, layout, accessibility, and user engagement. By redesigning the website, I aimed to enhance user satisfaction and make it easier for visitors to find information, access library resources, and engage with the community.


Research:


I began by analyzing the existing website of the Lower Providence Community Library to identify usability problems and areas for improvement. Through this analysis, I found several issues related to efficiency, learnability, layout and navigation, memorability, and accessibility. I also considered feedback from potential users, who highlighted the importance of easy navigation, clear information presentation, and accessibility features. The audience for this redesign includes library patrons of all ages who visit the website to access library resources, find information about events, and engage with the community.


Process:


Identifying Usability Problems

I analyzed the original website to identify usability problems such as inefficient navigation, unclear layout, and accessibility issues. Problems included a sliding gallery for new releases, unclear event organization, and difficulty registering for a library account.


Sketches:

I performed speed sketching to brainstorm layout options, leading to the creation of wireframes for mobile, tablet, and desktop screens. The wireframes addressed issues like navigation placement, button consistency, and event organization.


High-Fidelity Mockups:

Based on wireframes, I designed high-fidelity mockups for mobile, tablet, and desktop screens, focusing on clarity, consistency, and accessibility. Mockups showcased improved navigation, clearer presentation of information, and enhanced visual appeal.


Visual Redesign

Using HTML and CSS, I developed the responsive final website, ensuring compatibility with different devices and screen sizes. Accessibility features were incorporated, and usability testing was conducted to refine the design. The website passed accessibility checks with zero errors and zero alerts, demonstrating its improved usability and accessibility.


Solution:


The redesigned Lower Providence Community Library website addresses usability problems, enhances user experience, and improves engagement. Key features of the solution include: Improved navigation with a fixed top menu for easy access to important sections. Clear presentation of information, including upcoming events, reading challenges, and account registration. Consistent layout and design across different devices, ensuring a seamless user experience. Enhanced accessibility features, such as proper contrast, aria labels, and responsive design. The impact of the redesign is reflected in increased user satisfaction, improved accessibility, and higher engagement with library resources and events.


Learnings and Conclusion:


Iterative design process: Iterating on wireframes and mockups helped identify and address usability issues effectively. Importance of accessibility: Incorporating accessibility features from the beginning ensures a more inclusive user experience. User-centered design: Considering user feedback and preferences is crucial for creating a user-friendly interface. Responsive design: Designing for multiple devices requires careful consideration of layout, navigation, and content presentation.
