Bucket List


Allowing Users to Plan Trips and Vacations



Context and Overview:


The goal of this project was to understand and apply different statistical tests used in design analysis to evaluate the impact of design changes on user interaction. By conducting A/B testing, I aimed to identify which design choices led to improvements in user experience and engagement on a web page. This project is crucial for designers and developers to make informed decisions and create interfaces that effectively meet user needs.


Research:


The research involved understanding the principles of A/B testing and statistical analysis to evaluate the effects of design changes on user interaction. The audience includes designers, developers, and product managers who are interested in optimizing user experience and engagement on digital platforms. They seek insights into how design choices impact user behavior and how to improve usability and efficiency.


Process


Data Collection

I was provided with a simple webpage, MedX, and tasked with scheduling an appointment, then I made UI changes to improve interaction. Changes included darkening button colors, adding spacing between appointment slots, and organizing appointments chronologically.


Analysis

I created null and alternative hypotheses for misclick rate, time on page, and time to first click. Hypotheses were based on expected improvements in user interaction due to design changes.


Predictions about Rejecting the Null Hypothesis

Predicted rejection of null hypotheses based on observed differences between versions A and B. Anticipated improvements in misclick rate, time on page, and time to first click for the modified website.


Statistical Tests

Conducted chi-squared test for misclick rate and one-tailed t-tests for time on page and time to first click. Analyzed key metrics including degrees of freedom, t-scores, and p-values


Solution and Impact:


Misclick Rate: Original Website: 30% Modified Website: 15% Decreased misclicks by 50%, indicating improved button visibility and clarity. Time Spent on Page: Original Website: 24,398.29 milliseconds Modified Website: 7,714.48 milliseconds Reduced average time on page by approximately 68%, indicating increased efficiency and user engagement. Time to First Click: Original Website: 12,217.43 milliseconds Modified Website: 4,803 milliseconds Reduced average time to first click by approximately 61%, indicating enhanced user engagement and usability.


Learnings and Insights


A/B testing is a powerful method for evaluating design changes and improving user experience. Statistical tests like chi-squared and t-tests provide valuable insights into the effectiveness of design choices. Design decisions should be informed by data and user behavior to optimize usability and engagement.
